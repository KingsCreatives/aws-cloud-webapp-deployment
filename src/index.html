Welcome to Your AWS Cloud Deployment

This application is securely delivered through Amazon CloudFront, routed using an Application Load Balancer, hosted on Amazon EC2, and built following AWS Well-Architected best practices.

⚖️ Load Balanced

Application Load Balancer distributes traffic across healthy EC2 instances for maximum availability.

🌍 CDN Enabled

Amazon CloudFront caches content at edge locations worldwide for low latency and faster delivery.

🚀 CI/CD Ready

GitHub Actions automates testing and deployment for reliable software delivery.
